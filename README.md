# Beesoft Components

Beesoft Components is a collection of components that can be used to manage data, for navigation, build forms, and mobile applications.

The current list of components are

* Form
  * Date/Time
  * Inputs
    * Content Editable Input
    * Formatted Input
* Mobile
  * Overlay
* Navigation
  * Buttons
    * Button
* Overlay
  * Overlay Panel
